### Closest 3

**[iBotAssistant](https://www.mdpi.com/1424-8220/26/6/1958).** For pillar (a), it combines Pupil Core gaze and speech to distinguish a specific piece of lumber, although the reported examples are “get me the lumber” rather than an explicit “this” construction. For pillar (b), gaze selects an AprilTag ID in the wearer’s 2D POV; it does not localize the wearer or intersect a gaze ray with a persistent global 3D instance map. For pillar (c), a real Husky–UR5e autonomously travels to the storage zone, picks the specified lumber, returns, and delivers it. However, the paper reports one shared laboratory with two zones—not different rooms—and does not establish that the referent was never in the robot’s FOV during the interaction.

**[FAM-HRI](https://arxiv.org/abs/2503.16492).** Pillar (a) is strongly satisfied: Meta Aria supplies wearable gaze and speech, with temporal gaze-word alignment resolving referenced objects. Pillar (b) is explicitly not satisfied because the system matches the wearer-view target to candidates in the robot’s current RGB-D view. Its ORB-SLAM3 trajectory processing is not a persistent shared semantic map that yields a referent’s global coordinate. Pillar (c) is absent: execution is by a stationary tabletop Panda, with no navigation or fetch.

**[DynaMem](https://arxiv.org/abs/2411.04999).** Pillar (a) is absent: the input is an explicit language object query, not wearable-gaze deixis. Pillar (b) is its strength—a long-lived dynamic 3D semantic memory supports reasoning about objects beyond the robot’s current camera view, although the representation is voxelized point-cloud memory rather than 3DGS. Pillar (c) is partially satisfied through real mobile navigation and pick/drop execution on Stretch. It never demonstrates the claimed interaction geometry: a wearer gazing at a referent while the robot is in another room and receiving that referent’s map-derived world coordinate.

### Verdict

CLAIM SURVIVES — as of the live sweep through 19 July 2026, no verified work combines (a) spoken wearable-gaze deixis, (b) gaze-ray grounding through wearer pose into a persistent metric global instance map while the referent is outside the robot’s view, and (c) autonomous cross-room mobile fetch. The dangerous prior art is iBotAssistant, so novelty should be asserted only for that full conjunction—not broadly as the first gaze-and-speech mobile manipulation system. This is a literature-search verdict, not proof against unpublished systems or unindexed patent material.
