### Closest 3

1. **[FAM-HRI](https://arxiv.org/abs/2503.16492).** For pillar (a), it is the strongest match: Meta Aria supplies live gaze and speech, with language conveying the task and gaze disambiguating the referent. For pillar (b), ORB-SLAM3 only stabilizes the wearer-side observation; feature matching must subsequently find the selected instance in the robot’s current RGB-D image, so the referent is not outside robot view and there is no persistent global instance map. For pillar (c), execution is stationary Franka tabletop manipulation, with no navigation or cross-room fetch. It anticipates the interaction logic but not the shared-map-mediated remote execution.

2. **[Gaze2Act](https://arxiv.org/abs/2605.30282).** For pillar (a), it closely implements language-for-task and glasses-derived gaze-for-target, including object- and part-level disambiguation. For pillar (b), it bridges ego and robot views through semantic correspondence into the robot’s current observation, rather than intersecting a gaze ray with a persistent global map; therefore the target must be robot-visible. For pillar (c), its Unitree G1 performs substantial manipulation, but the reported tasks are local and show no cross-room navigation or fetch. It is the strongest modern policy-level near miss.

3. **[EgoSpot](https://arxiv.org/abs/2306.02393).** For pillar (a), it has wearable gaze and voice, but voice controls modes while gaze selects a spatial point—not a spoken deictic object reference such as “this cup.” For pillar (b), ray–world-mesh intersection and Azure Spatial Anchor co-localization do transmit a metric target to Spot, but there is no persistent semantic object-instance map or demonstration involving an object outside the robot’s view. For pillar (c), Spot is precisely the relevant quadruped-plus-arm embodiment and walks to gaze-selected targets, yet the evaluation stops at locomotion and touching rather than autonomous grasping, cross-room retrieval, and delivery. This is the closest hardware/geometric precursor, but still misses each decisive qualification.

### Verdict

**CLAIM SURVIVES** — No verified public work found through 2026-07-19 demonstrates all three pillars in one end-to-end system, especially the decisive condition that a gaze-resolved instance absent from the robot’s view is retrieved through a shared persistent global map. Existing work separates into wearable speech–gaze grounding with robot-visible tabletop objects, wearable metric-point control of mobile robots, gaze-aware 3D datasets/maps without execution, and mobile fetch without wearer-gaze deixis; this remains a literature-search conclusion rather than proof against unpublished systems.
